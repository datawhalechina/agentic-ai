# Agentic AI 课程笔记

本项目围绕吴恩达老师在 DeepLearning.AI 推出的 Agentic AI 系列课程，致力于为中文学习者提供高质量的课程内容翻译、系统化的知识梳理、关键概念解析以及配套示例代码的详细解读。

**在线视频课程地址：** [DeepLearning.AI - Agentic AI](https://learn.deeplearning.ai/courses/agentic-ai/)

## 课程目录

| 章节 | 内容 |
| --- | --- |
| Module 1 | Agentic工作流简介 (Introduction to Agentic Workflows) |
| Module 2 | 反思设计模式 (Reflection Design Pattern) |
| Module 3 | 工具使用 (Tool Use) |
| Module 4 | 构建Agentic AI的实用技巧 (Practical Tips for Building Agentic AI) |
| Module 5 | 高度自治智能体的模式 (Patterns for Highly Autonomous Agents) |

## 项目受众

- 对 Agentic AI 感兴趣
- 正在系统学习 LLM 高级应用
- 希望掌握 LLM 与本地工具集成

- 关注 AI 自动化流程与任务协作系统
