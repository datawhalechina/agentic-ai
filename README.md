## FAQ

### 什么是 Agentic AI？

Agentic AI（智能体 AI）是一种 AI 系统设计模式，将大语言模型（LLM）作为核心"智能体"，赋予其自主规划、决策和执行任务的能力。与传统 AI 应用不同，Agentic AI 强调：
- **自主性**：Agent 能够自主分解复杂任务，规划执行步骤
- **工具使用**：Agent 可以调用外部工具（API、数据库、文件系统等）
- **反思迭代**：Agent 能够评估自身输出，进行自我改进
- **多智能体协作**：多个 Agent 可以协同完成复杂任务

| Module | 主题 | 关键知识点 |
|--------|------|-----------|
| Module 1 | Introduction to Agentic Workflows | Agent 基础概念、Agentic Workflow 设计模式 |
| Module 2 | Reflection Design Pattern | 自反思机制、迭代优化策略 |
| Module 3 | Tool use | 工具调用设计、Function Calling 实践 |
| Module 4 | Practical Tips | 构建 Agent 的最佳实践、调试技巧 |
| Module 5 | Patterns for Autonomous Agents | 高度自主 Agent 模式、多 Agent 协作 |

### 如何开始学习？

推荐学习路径：
1. **观看原课程视频**：在 [DeepLearning.AI](https://learn.deeplearning.ai/courses/agentic-ai/) 观看吴恩达老师的课程
2. **阅读中文翻译**：参考本项目的翻译内容，深入理解技术细节
3. **实践代码示例**：运行配套示例代码，掌握实现方法
4. **参与社区讨论**：在 Issues 中提问、分享学习心得
5. **贡献改进**：提交翻译改进、补充笔记或新增示例

- 具备基础 Python 编程能力的开发者

- 关注 AI 自动化和任务协作系统的技术人员
